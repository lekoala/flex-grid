Setting gutters

Gutters are calculed either with negative margin or a mix of horizontal padding and negative vertical margin. This is controlled by the $enable-gutters-padding. By default, we use negative margin.

Use padding if you don't like hiding overflow and use negative margin if you don't like nesting tags. "unstyled" grids in demo should display the same regardless of the setting. You can see they require additionnal "demo-div" to show the grid.

Use the g global attribute or dedicated gx and gy attributes

Use flex-col

Instead of regular divs, you can use a flex-col element. They allow setting individual column settings through attributes

col 1 with 100px col 2 goes first col 1 end col 2 start col 1 is only visible on mobile col 2 is always visible

Advanced flex-container

It can add padding if necessary with pad. You can pass an optional "dynamic" value which doubles the padding starting from md displays.
